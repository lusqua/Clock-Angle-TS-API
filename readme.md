# This is a clock angle challenge in TypeScript
## How to run

1. Install typescript

```bash
npm install -g typescript
```

2. Install the project dependencies

Usining npm
```bash
npm install
```

or using yarn
```bash
yarn install
```

3. Run the project

Usining npm
```bash
npm run start
```
or using yarn
```bash
yarn start
```

## How to test

Using npm
```bash
npm run test
```
or using yarn
```bash
yarn test
```
